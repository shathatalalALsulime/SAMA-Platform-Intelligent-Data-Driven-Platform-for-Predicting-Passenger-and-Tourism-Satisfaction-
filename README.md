# SAMA Platform – Intelligent Data-Driven Platform for Predicting Passenger and Tourism Satisfaction
graduation project for airline operations analytics and predictive modeling

## Project Overview
This project presents an intelligent data-driven platform designed to analyze passenger feedback, measure service performance, and predict passenger satisfaction within the tourism and airline industry.

The platform combines data preprocessing, sentiment analysis, KPI evaluation, and machine learning prediction models to generate insights from airline passenger reviews and operational data.

The project was developed using Python and several data science and machine learning libraries within Google Colab notebooks.
