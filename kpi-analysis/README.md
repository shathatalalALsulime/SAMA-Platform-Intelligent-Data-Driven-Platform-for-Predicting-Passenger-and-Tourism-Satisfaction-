# KPI Analysis

## Overview
This notebook focuses on analyzing airline operational performance and customer experience KPIs using structured aviation and passenger datasets.

The analysis evaluates:
- Airline operational efficiency
- Passenger satisfaction
- Delay performance
- Service quality
- Complaint patterns
- Seasonal operational impact

The notebook combines descriptive analytics, KPI engineering, correlation analysis, and business intelligence metrics to generate operational insights for airline performance evaluation.

---

## Dataset Information

The dataset contains airline operational and passenger service records including:
- Flight status
- Delay information
- Passenger satisfaction
- Operational efficiency
- Service ratings
- Waiting times
- Complaints
- Weather conditions
- Seasonal event data

Main columns used:
- Airline_Name
- Flight_Status
- Delay_Minutes
- Satisfaction_Index
- Operational_Efficiency_Score
- Checkin_Time_Minutes
- Boarding_Time_Minutes
- Baggage_Claim_Time_Minutes
- Onboard_Service_Rating
- Complaints_Logged
- Event_Season
